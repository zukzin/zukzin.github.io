This is unit 9 formative activity. The topic here is Quality Management Strategies appreciating the range of ways through which software quality may be determined to have been achieved.

Unit 9: Improving Code Quality

Refer to the Mertz (2019) resource. Use some Python code which you have developed in the past and apply at least 3 of the strategies presented at the source to improve its quality. You can use the Jupyter Notebook workspace and save your work to your GitHub repository.

RESPONSE:

Common strategies for improving Python Code. The following are based on typical best practice from Python Literature

  1. Refactoring: Simplifying the code structure to improve readability and reduce redundancy.
  2. Error handling: Implementing proper error and exception handling.
  3. Code efficiency: Optimizing loops, functions, or libraries to improve performance.
  4. Testing: Adding unit tests to ensure the code works as expected.
  5. Modularity: Breaking down complex code into smaller, reusable functions.

Based on strategies from Mertz (2019), apply at least three enhancements to the code.

  • Refactor to make it more concise.
  • Add error handling to ensure robustness.
  • Improve variable naming for better readability.

To execute the code in a Jupyter Notebook to ensure it still works after the improvements.

 #original code
def calculate_total(price, tax_rate):
total = price + (price * tax_rate)
return total

Using improved based on the three strategies mentioned above


def calculate_total_cost(item_price, tax_percentage):
    """
    Calculate the total cost including tax.

    Args:
        item_price (float): The price of the item.
        tax_percentage (float): The tax rate as a decimal.

    Returns:
        float: The total cost including tax.
    """
    try:
        if item_price < 0 or tax_percentage < 0:
            raise ValueError("Price and tax rate must be non-negative.")
        
        total_cost = item_price + (item_price * tax_percentage)
        return round(total_cost, 2)
    
    except TypeError as e:
        print(f"Type Error: {e}")
    except ValueError as e:
        print(f"Value Error: {e}")
