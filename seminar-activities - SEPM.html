Unit 2 Seminar: Requirements Gathering

Read the Behaviour Driven Development (2020) pages and then use the Gherkin language to create a Gherkin sequence that addresses ONE of the following examples:

  • Using a new coffee making machine.
  • Interfacing with a new SatNav system.
  • Using a computer running the Linux operating system.
  • Getting familiar with a new vehicle.
  • Creating a batch or shell script.

    • Your response should consist of at least three scenarios describing different roles such as administrator, user, driver and so on. Be prepared to share your scenarios at this week's seminar. You can test your Gherkin scripts in the Behave engine.

    RESPONSE

    Interfacing with a new SatNav system scenario 

     from behave import given, when, then

		# Scenario 1: Driver sets up destination
		@given('the SatNav system is powered on')
		def step_impl(context):
			context.satnav_powered_on = True
			print("SatNav system powered on.")

		@given('the driver is seated in the vehicle')
		def step_impl(context):
			context.driver_seated = True
			print("Driver is seated.")
										
		@when('the driver inputs the destination address')
		def step_impl(context):
			if context.satnav_powered_on and context.driver_seated:
				context.destination_set = True
				print("Destination address inputted.")
			else:
				context.destination_set = False
				
		@then('the SatNav system should calculate the best route')
		def step_impl(context):
			if context.destination_set:
				context.route_calculated = True
				print("Best route calculated.")
		   	else: 
				context.route_calculated = False
										
		@then('the system should display the estimated time of arrival')
		def step_impl(context):
		    	if context.route_calculated:
				print("Estimated time of arrival displayed.")
										
										
		# Scenario 2: System Administrator updates the SatNav software
		
		@given('the SatNav system is connected to the internet')
		def step_impl(context):
				context.internet_connected = True
				print("SatNav connected to the internet.")
										
		@given('the System Administrator has the latest software update file')
		def step_impl(context):
				context.update_file_ready = True
				print("Software update file ready.")
										
		@when('the System Administrator initiates the update')
		def step_impl(context):
			if context.internet_connected and context.update_file_ready:
				context.update_initiated = True
				print("Update initiated.")
			else:
				context.update_initiated = False
										
		@then('the SatNav system should install the update')
		def step_impl(context):
			if context.update_initiated:
				context.update_installed = True
				print("Update installed.")
			else:
				context.update_installed = False
										
		@then('the system should restart to apply the changes')
		def step_impl(context):
			if context.update_installed:
				print("System restarting to apply changes.")
										
		# Scenario 3: Passenger checks for alternative routes
		@given('the SatNav system is currently navigating to a destination')
		def step_impl(context):
			context.navigating = True
			print("SatNav currently navigating.")
										
		@given('there is traffic on the current route')
		def step_impl(context):
			context.traffic_detected = True
			print("Traffic detected on the current route.")
										
		@when('the passenger selects the "Alternative Routes" option')
		def step_impl(context):
			if context.navigating and context.traffic_detected:
				context.alternative_routes_available = True
				print("Passenger selected 'Alternative Routes' option.")
			else:
				context.alternative_routes_available = False
										
		@then('the SatNav system should display a list of alternative routes')
		def step_impl(context):
			if context.alternative_routes_available:
				print("Displaying list of alternative routes.")
										
		@then('the passenger can select the preferred route')
		def step_impl(context):
			if context.alternative_routes_available:
				print("Passenger selected preferred route.")
